Visualisation Mathématique

L’idée est de tracer la fonction :

π(x) = nombre de nombres premiers ≤ x

C’est une fonction en escalier :

  • Elle reste constante entre deux nombres premiers,
  • Elle augmente de 1 à chaque fois qu’on rencontre un nombre premier.

Exemple :

  • π(2) = 1
  • π(3) = 2
  • π(4) = 2
  • π(5) = 3
  • π(6) = 3
  • π(7) = 4

Graphiquement, c’est une suite de marches : un escalier irrégulier qui monte de 1 à chaque nombre premier.
